# diversity-weighted-portfolio

基于 Fernholz 随机投资组合理论 (Stochastic Portfolio Theory) 的多样性加权投资组合策略的编程复现与实现。

## 理论背景
本项目复现了 E. Robert Fernholz 在 *Stochastic Portfolio Theory* 中提出的核心概念：**多样性加权 (Diversity-Weighted) 投资组合**。

与传统的市值加权 (Capitalization-Weighted) 组合不同，该策略通过对股票权重进行“多样性”变换，数学上可证明其能够产生**超越基准**的长期收益，且在波动率上具有天然优势。
